# CareBridge – Smart Health Companion for Seniors

CareBridge is a web application designed to help elderly individuals manage their health while allowing caregivers to monitor their wellbeing remotely.

The system provides medication reminders, health tracking, appointment management, and an emergency SOS alert system that instantly notifies caregivers.

---

# Features

## Elder Features
- Medication reminders with alarm
- Health logs tracking
- Health graphs visualization
- Appointment management
- Emergency SOS alert button
- Simple and accessible interface

## Caregiver Features
- Real-time SOS alerts
- Monitor elder health logs
- Track medication status
- View elder health history

# Installation

Clone the repository
git clone https://github.com/akshar-J11/CareBridge-Smart-Health-Companion-for-Seniors.git


Go to the project folder
- cd CareBridge-Smart-Health-Companion-for-Seniors

Install dependencies
- npm install

Run the development server
- npm run dev


Open the browser
- http://localhost:5173
